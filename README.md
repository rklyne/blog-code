
# Blogging platform

## Running it

run `./blog.py regenerate` to generate a new set of html files

## Writing new entries

Add new post files to data/posts/<name>.md.
Those files are all of the form:
    #!param=value
    ## Markdown code
    And the text...

## Parameters

 * `name` - a title for the post
 * `desc` - a paragraph to describe the post.
